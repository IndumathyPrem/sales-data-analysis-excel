# 📊 Sales Data Analysis & Dashboard (Excel)

## 📌 Quick Summary
An Excel-based project where raw sales data was cleaned, validated, and transformed into an interactive dashboard for business insights.

---

## 🎯 Problem Statement
Analyze sales data to identify trends, fix data issues, and generate actionable insights.

---

## 🛠️ Tools & Skills
- Microsoft Excel
- Data Cleaning (CLEAN, TRIM, PROPER)
- Pivot Tables
- Data Validation
- Dashboard Design

---

## 📁 Project Structure
- Dataset → Raw & Cleaned Data
- Dashboard → Final Excel Dashboard
- Screenshots → Dashboard Preview

## ⭐ Key Features
- Cleaned and standardized dataset
- Fixed missing & incorrect values
- Built interactive dashboard
- Generated business insights

---

## 📈 Key Insights
- North & South regions generate highest revenue
- Loyal customers contribute most sales
- West region underperforms

---

## 💡 Business Recommendations
- Improve inventory tracking
- Focus on loyal customers
- Target low-performing regions

---

## ⚙️ How to Use
1. Download the Excel file
2. Open in Excel
3. Go to Dashboard sheet
4. Use slicers to explore data
